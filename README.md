# 抖音私信自动回复助手

这是一个基于Python的抖音私信自动回复工具，使用Coze AI提供智能回复功能。

## 主要功能

1. **抖音账户操作**
   - 账户链接互动
   - 视频点赞、收藏、评论

2. **视频互动**
   - 简单评论模式
   - 评论区评论模式

3. **私信自动回复**
   - AI智能回复（基于Coze API）
   - 自定义消息回复
   - 多种回复风格选择

## 使用方法

### 安装依赖

```bash
pip install helium requests
```

### 运行程序

```bash
python main.py
```

### 私信回复功能使用步骤

1. 在主菜单选择"1.抖音"
2. 选择"3.私信自动回复"
3. 登录抖音账号
4. 配置回复设置（可选）
5. 选择"1.开始监控私信"

## AI回复风格设置

提供四种回复风格：
- 高冷女主播
- 热情活泼
- 商务专业
- 自定义风格

## Coze API设置

可以在菜单中更新Coze API凭证：
- Coze令牌
- 机器人ID

## 独立测试工具

可以使用独立的Coze客户端测试API连接：

```bash
python -m utils.coze_client --debug
```

## 注意事项

- 确保网络良好，以便于与抖音和Coze API的通信
- 使用前需要有一个可用的抖音账号
- Coze API需要有效的令牌和机器人ID
